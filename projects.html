PG Project

A Combined Approach for Robust Face Recognition System in an Unconstrained Environment

Image Processing (MATLAB)

Face recognition system is one of the prominent research areas in pattern recognition and computer vision because of its major challenges. Some of the challenges in recognizing faces are blur, illumination, and varied expressions. To solve these issues, facial expressions are removed from face images using a pre-processing technique and Blur and Illumination-Robust Face recognition algorithm was applied to identify the person.

UG Project

Towards A Practical Event-Based Information Refinding System

ASP.NET, SQL Server 2008

Refinder system refinds web pages and local files in our system. It leverages human’s natural recall characteristics and allows users to re-find files and Web pages according to the previously accessed context. Refinder refinds information based on a query-by-context model over a context memory snapshot, linking to the accessed information contents. The events in the memory snapshot are organized in clustered and associated manner.
